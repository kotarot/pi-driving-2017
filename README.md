# 議事録


##2017年4月25日

PS4のコントローラーのデータを取得した。
Pygameライブラリを用いる。
サンプルプログラム (joystick.py)[/joystick.py] を参考に
[drive.py](/drive.py) を作成した。

コントローラの接続のやり方:
```
sudo ds4drv
```
ここでSHAREボタンとPSボタンを長押しする（白いライトが点滅する）。
接続後、Pythonプログラムを実行する。
```
python3 drive.py
```
